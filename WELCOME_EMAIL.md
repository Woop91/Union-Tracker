Hi there,

Thanks for checking out this project! I created this system as a personal side project to help chapters, groups, or organizations keep track of:

• Membership contact information – A centralized directory for managing member details
• Grievance status tracking – Monitor cases, deadlines, and outcomes in one place
• Survey data presentation – Collect and visualize member feedback and satisfaction metrics

The system is designed to be customized to fit different chapters, groups, or organizations—you can adapt the terminology, deadlines, workflows, and structure to match your specific needs.

IMPORTANT NOTE: All member data currently shown in the forms and sheets has been randomly generated for demonstration purposes and is NOT based on real members or work locations.

───────────────────────────────────────────────────────────────────────────────
WHAT THIS SYSTEM DOES
───────────────────────────────────────────────────────────────────────────────

• Member Directory - Central hub for all member information
• Grievance Tracking - Automatic deadline calculations and reminders
• Satisfaction Surveys - Track and analyze member feedback
• Dashboards - Visual analytics for members and stewards
• Google Drive Integration - Automatic folder creation for each case
• Email Automation - Deadline reminders and status updates


───────────────────────────────────────────────────────────────────────────────
HOW INFORMATION FLOWS
───────────────────────────────────────────────────────────────────────────────

1. Members receive Contact Form link → Update their own information
2. Information populates Member Directory → Centralized database
3. From Member Directory, stewards can:
   - Send satisfaction survey links
   - Send personalized dashboard links
   - Start a grievance (pre-populates member info)
4. Steward completes Grievance Form → Only adds complaint details
5. Grievance Log auto-populates → Calculates deadlines, creates Drive folder
6. Evidence Collection → Upload documents to linked Google Drive folder


───────────────────────────────────────────────────────────────────────────────
KEY FEATURES
───────────────────────────────────────────────────────────────────────────────

GRIEVANCE LOG
• Automatic deadline calculations based on your contract
• Traffic light indicators (Red = Overdue, Orange = Urgent, Green = On Track)
• Google Drive folder auto-creation for each case
• Email reminders sent to stewards

MEMBER DASHBOARD (For All Members)
• Union snapshot and aggregate statistics
• Steward directory with contact info
• Member satisfaction results (aggregated)
• Weingarten Rights quick reference
• Grievance statistics and win rates

STEWARD DASHBOARD (For Stewards Only)
• Overview: Total members, active grievances, win rate
• Workload: Case distribution and capacity indicators
• Analytics: Trends and patterns
• Hot Spots: Locations with multiple grievances
• Bargaining: Contract violation tracking
• Satisfaction: Member feedback analysis
