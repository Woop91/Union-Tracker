Usage: Getting Started

From zero to fully operational in four steps

1️⃣

Clone & Install

Clone the repo and run npm install to set up the development environment.

2️⃣

Configure

Fill in the Config sheet: job titles, offices, units, supervisors, stewards.

3️⃣

Build & Deploy

Run npm run deploy to build, test, and push to Google Apps Script via clasp.

4️⃣

Open & Use

Open the Sheet — menus auto-populate via onOpen(). Start adding members and grievances.
