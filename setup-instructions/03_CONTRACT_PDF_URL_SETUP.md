# PDF LINK

## Contract PDF URL Setup Instructions

### Step 1: Open your Config sheet

Navigate to the Config sheet in your 509 Dashboard spreadsheet.

### Step 2: Find column AZ

Scroll right to column AZ (after column AY "PDF_FOLDER_ID").

### Step 3: Add header (row 1)

In cell **AZ1**, enter:

```
Contract PDF URL
```

### Step 4: Add your contract URL (row 2)

In cell **AZ2**, paste your contract PDF link.

**Accepted URL formats:**

| Type | Example |
|------|---------|
| Google Drive | `https://drive.google.com/file/d/1ABC123xyz/view` |
| Direct PDF | `https://yoursite.org/contracts/2023-2026-CBA.pdf` |
| Shared Drive | `https://drive.google.com/open?id=1ABC123xyz` |

### Step 5: Test it

Open any dashboard (Steward Dashboard, Member Portal, or Public Portal) and click the **"Union Contract"** or **"Contract"** button to verify the link works.

---

> **Note:** If the cell is left empty, the contract button will show `#` (non-functional link) until configured.
