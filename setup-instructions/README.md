# Setup Instructions

This folder contains setup instructions for various features of the 509 Strategic Command Center.

## Contents

| File | Description |
|------|-------------|
| [01_HEATMAP_SETUP.md](01_HEATMAP_SETUP.md) | Heatmap color gradient configuration for Google Sheets |
| [02_OCR_SETUP.md](02_OCR_SETUP.md) | Google Cloud Vision API setup for OCR functionality |
| [03_CONTRACT_PDF_URL_SETUP.md](03_CONTRACT_PDF_URL_SETUP.md) | Contract PDF link configuration in Config sheet |
| [04_RESOURCE_DRIVE_FOLDER_SETUP.md](04_RESOURCE_DRIVE_FOLDER_SETUP.md) | Shared documents folder setup in Google Drive |
| [05_PDF_GENERATION_SETUP.md](05_PDF_GENERATION_SETUP.md) | Grievance PDF generation and email setup |

## Prerequisites

Before following these instructions, ensure you have:

- Access to the 509 Dashboard Google Spreadsheet
- Appropriate permissions (Owner or Editor)
- A Google Cloud account (for OCR setup)
- Access to Google Drive for file storage

## Order of Setup

For a complete setup, follow the instructions in this order:

1. **Resource Drive Folder** - Set up shared documents first
2. **Contract PDF URL** - Add your union contract link
3. **PDF Generation** - Configure grievance PDF templates
4. **OCR Setup** - Enable OCR if needed for handwritten forms
5. **Heatmap** - Apply conditional formatting as desired
