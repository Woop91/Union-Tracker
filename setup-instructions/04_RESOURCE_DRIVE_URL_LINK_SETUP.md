Instructions: Setting Up the Shared Docs Folder
Step 1: Create the Google Drive Folder
Go to Google Drive
Click "+ New" → "New folder"
Name it something like "Member Resources" or "Shared Documents"
Click Create
Step 2: Set Sharing Permissions
Right-click the folder → "Share"
Under "General access", change to:
"Anyone with the link" (if members just need view access)
Or add specific email addresses for restricted access
Set permission level to "Viewer" (recommended) or "Editor"
Click Done
Step 3: Get the Folder ID
Open the folder in Google Drive
Look at the URL in your browser:
https://drive.google.com/drive/folders/1AbCdEfGhIjKlMnOpQrStUvWxYz123456

Copy the ID portion (everything after /folders/):
1AbCdEfGhIjKlMnOpQrStUvWxYz123456

Step 4: Add to Config Sheet
Open your Google Spreadsheet
Go to the Config sheet
Find Column Y (labeled "DRIVE_FOLDER_ID" in the Integration section)
Paste the folder ID into that cell
Step 5: Verify It Works
Open a member dashboard
Look for the "Resources" button (green button with folder icon)
Click it - should open your shared folder
What to Put in the Folder
Consider adding:

Union bylaws/constitution
Contract/CBA documents
Meeting minutes
Training materials
Forms and templates
Contact information sheets
